# C++ Raytracing 
## How to compile
from project root
```
mkdir build
cd ./build
cmake ..
cmake --build .
```
## How to run 
from project root
```
cd ./build/Debug
./CSE3PSD_Assignment_3_21985164.exe height (in px) width (in px) anti-aliasing (0 = off 1 = on)
```
### example usages 
```
./CSE3PSD_Assignment_3_21985164.exe 1080 1080 1
```
products a 1080x1080 image with anti aliasing on
